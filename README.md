# FileMind AI Agent

A local AI-powered file system assistant built using:

- Python
- Ollama
- Llama 3.2
- SQLite
- PyMuPDF
- Rich

## Features

- Natural language file search
- PDF summarization
- Duplicate file detection
- Local execution
- Command history
- SQLite memory

git clone https://github.com/yourusername/filemind-ai-agent.git
